  • List what inside directory
    $ ls
    $ ls -a
    $ ls -l
  • Make new directory
    $ mkdir
  • Change to directory
    $ cd [directory]
    Current directory (.)
    $ cd .
    Parent directory (..)
    $ cd ..
  • Print working directory
    $ pwd
  • Copy file (makes a copy of file1 in the current working directory and calls it file2)
    $ cp file1 file2
    // copy file to current directory:
    $ cp /home/hungpham/something.txt .
    // copy folder:
    $ cp -a /source/. /dest/
